# EventList

_Self hosted linktree alternative_

## Prerequisites

1. `brew install python3`
2. put your events into `events.json`
3. put your social links into `socials.json`

## How to use

1. run `serve.sh` (starts a local python web server)
2. `serve.sh` should open the landing page in your default browser

> if it doesn't open automatically, use the `Local Url` generated by `serve.sh`

### if serve.sh does not work (only tested on macOS)

1. run the following command in the `EventList` directory:

   ```
   python3 -m http.server 48080
   ```

2. use the following url:

   ```
   http://localhost:48080/?q=displayAll
   ```
